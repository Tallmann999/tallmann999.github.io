Core Idea

Caliber of Chaos centers on elemental seals: enemies and objects are protected by elements that require matching weapon elements to break. This creates a combat-puzzle layer on top of fast shooter gameplay.

Core Gameplay Loop

1. Enter corrupted area
2. Identify seals & mutations
3. Switch weapon element
4. Break seals & damage
5. Clean objects / gain allies

Implemented Gameplay Systems

Elemental Combat System

Multiple element types (Bio Red, Crystal Blue, Purge Green, Gold Yellow) change weapon interactions and enemy reactions.

Seal System

Seals protect enemies and objects — break seals with matching elements to enable damage.

Enemy Behavior

Melee, shooter, patrol, elite and spawner logic with mutation states that alter patterns.

Mutation / Biome System

Zones modify ground, enemies and objects — Purge Green can clean and transform objects.

Environmental Allies

Transformable objects (trees, devices) become temporary allies to assist in combat.

Technical Challenge — Problem → Risk → Solution → Result

Problem

Bullet collision logic consolidated many responsibilities (damage, seals, mutation, faction, pooling), becoming hard to maintain.

Risk

Growing complexity slowed iteration and added bugs when adding new elemental effects.

Solution

Split bullet responsibilities: movement, visuals, collision processing, damage rules, seal interaction, and pool return logic.

Result

Cleaner, extensible bullet pipeline enabling fast iteration on elemental effects and fewer side-effects.
